# Lab Skills Reference

This folder contains structured skill files for each lab in the **Create Copilot Chatbots and Agents with Microsoft Copilot Studio** course.

## Skill File Structure

Each skill file follows a consistent format:

```markdown
# Lab Title

## Lab Title
[Full descriptive title]

## Lab Objectives
[5-6 numbered learning objectives]

## Prerequisites
[Required accounts, tools, completed labs]

## Step-by-Step Guide
[Detailed numbered steps organized by sections]

## Duration
[Estimated time to complete]

## Next Steps
[Link to the next lab]
```

## Usage

These skill files can be used as:

1. **Quick Reference**: Look up specific lab objectives and prerequisites
2. **Instructor Guide**: Follow the step-by-step instructions for demonstrations
3. **Self-Paced Learning**: Complete labs independently with structured guidance
4. **Review Material**: Refresh knowledge on specific topics

## Prerequisites Summary

### Day 1 Prerequisites
- Microsoft 365 account with Copilot Studio access
- Power Platform environment (Developer or Trial)
- Microsoft Teams access
- SharePoint site for lab exercises

### Day 2 Prerequisites
- All Day 1 prerequisites
- Completed Day 1 labs (especially Labs 6-10)
- Hiring Agent solution imported
- Sample data loaded in Dataverse
